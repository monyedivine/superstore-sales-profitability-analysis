#Superstore Sales & Profitability Analysis
P#roject Overview

This project analyzes Superstore sales and profitability performance using Tableau.

#The dashboard focuses on:

Sales and profit performance
Customer profitability
Regional analysis
Discount impact on profit
Sub-category anomalies
Tools Used
Tableau
Excel
#Key Insights
Furniture recorded the highest product cost
Higher discounts were associated with lower profits
The Central region underperformed compared to other regions
Labels, Paper, and Envelopes generated the highest profit margins
Tables, Supplies, and Bookcases showed weak or negative margins
